# 🖥️ Server Monitoring Platform (SMP)

Plataforma de monitorización de servidores desarrollada en Python orientada a administradores de sistemas.

## 🚀 Features

- Monitorización de CPU, RAM y disco
- Sistema de alertas automatizadas
- Logs de eventos
- Configuración flexible mediante archivos YAML

## 🛠️ Tecnologías

- Python
- psutil
- logging

## ▶️ Uso

```bash
pip install -r requirements.txt
python main.py
